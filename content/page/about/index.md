大家好，这里是刚刚雨纷纷，你可以以任意喜欢的方式叫我，只要我认得出来。这个 ID 是小时候滚键盘取的，等我反应过来应该取个正常 ID 之后已经用了几年了，索性懒得改了(笑

关于我：懒得做自我介绍，你可以期待在文章里找到自己感兴趣的话题，当然如果你能摸到这个站，本身应该对我有点了解就是了。

对，懒，我是一个非常懒的人，懒得做家务，懒得收拾，懒得社交，懒得出门。甚至懒得去记东西，我记不住大量常识，记不住刚结束的旅游去了哪，最重要的是连往事都遗忘得很快。不知道这遗忘速度是不是正常的，但我觉得不能再这样了：如果我总是把过去忘记，那不管活了多久，人生不还是只有当下这一点吗？

于是我觉得务必要记录自己的事。本来一直以来都是丢进清水河畔，但可怜的电大竟然把校友都切割了，正好趁此机会建一个自己的地盘，故有此站。
